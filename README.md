📊 Private Equity Fund Dashboard (Streamlit)

This project is an interactive Private Equity Fund Dashboard built using Python + Streamlit + Plotly, designed to simulate how real-world PE firms analyze fund performance, portfolio holdings, and capital flows.
It replicates institutional-level reporting aligned with US GAAP (ASC 946) and is inspired by KPMG illustrative financial statements

👉 The goal is simple:
Turn theoretical PE concepts into practical, visual analytics.

🎯 Key Features
🏠 Fund Overview
Total Partners’ Capital tracking
IRR (Since Inception)
Net Income & Unrealized Gains
Sector & Geography allocation


📊 Portfolio Analytics
Full portfolio holdings (Cost vs Fair Value)
Gain/Loss & Return % per investment
Treemap visualization (sector-wise exposure)
Filters: Sector | Geography | Investment Type



💰 P&L Analysis
Income vs Expense breakdown
Investment gains (realized & unrealized)
Waterfall chart for Net Income


🌊 Cash Flow Analysis
Investment purchases vs exits
LP contributions & distributions
Cash flow waterfall (Operating + Financing)



👥 Partners’ Capital
GP vs LP capital allocation
Capital movement (YoY)
Carried Interest (Performance Fee)
Net Income distribution



🛠 Tech Stack
Python
Streamlit – UI & dashboard framework
Pandas / NumPy – Data processing
Plotly – Interactive visualizations



📈 Core Private Equity Concepts Covered
IRR (Internal Rate of Return)
Unrealized vs Realized Gains
Capital Calls & Distributions
Carried Interest (GP Incentive)
NAV / Fair Value Accounting
Portfolio Allocation & Diversification
